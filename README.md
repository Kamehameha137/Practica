### Установка и запуск 

1. `git clone https://github.com/Kamehameha137/Practica`

Клиент:

2. `cd Practice1/client`

3. `npm ci`

4. `npm run build`

5. `npm run dev`

Сервер:

2. `cd Practice1/server`

3. `npm ci`

4. `npm start`
